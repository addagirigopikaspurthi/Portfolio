# Addagiri Spoorthi Portfolio

A fast static personal portfolio for Addagiri Spoorthi, prepared for Vercel deployment.

## Local preview

```bash
npm install
npm run build
npm start
```

The preview server prints a local URL, usually `http://127.0.0.1:4173`.

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Use the included defaults:
   - Build command: `npm run build`
   - Output directory: `dist`
4. Deploy.
